# 死锁代码

public class Main {

    private static Object lock1 = new Object();
    private static Object lock2 = new Object();

    public static void main(String[] args) {
       Thread thread1 = new Thread(new Runnable() {
           @Override
           public void run() {
               synchronized (lock1){
                   System.out.println("thread1 hoding lock1");
                   try {
                       Thread.sleep(100);
                   } catch (InterruptedException e) {
                       throw new RuntimeException(e);
                   }
                   System.out.println("thread1 wait lock1");
                   synchronized (lock2){
                       System.out.println("thread1 hoding lock2");
                   }
               }
           }
       });

       Thread thread2 = new Thread(new Runnable() {
           @Override
           public void run() {
               synchronized (lock2){
                   System.out.println("thread2 hoding lock2");
                   try {
                       Thread.sleep(100);
                   } catch (InterruptedException e) {
                       throw new RuntimeException(e);
                   }
                   System.out.println("thread2 wait lock1");
                   synchronized (lock1){
                       System.out.println("thred2 holding lock1");
                   }
               }
           }
       });

       thread1.start();
       thread2.start();
    }

    public static void test(){
        System.out.println("test");
    }
}

# 定位死锁

  1. 使用jps -l命令，查询到所有java进程
  2. 找到自己程序的进程id
  3. 使用 jstack 进程id | grep 'deadlock' -A 100 查询死锁日志前100行
jps -l
jstack pid | grep 'deadlock' -A 100 # deadlock是死锁查询关键字
